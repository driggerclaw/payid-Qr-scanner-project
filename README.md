# PayID QR Scanner Feature

## Description
This feature allows users to scan a QR code to automatically fill PayID details.

## User Story
As a user, I want to scan a QR code so that I can make payments quickly.

## Features
- QR scanning
- Auto-fill PayID
- Error handling
## Validation
- Ensure QR contains valid PayID
- Show error if invalid
## QR Validation Rules
- Ensure the QR code contains valid PayID data
- Display an error message if the QR code is invalid
- Prevent incomplete payment details from being submitted
